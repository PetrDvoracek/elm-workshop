# Elm Workshop

## Install Elm compiler

1. Installer (Windows, macOS, Linux)
    - https://elm-lang.org

2. NPM package

        npm install -g elm

## Tools

1. elm-live

        npm install -g elm-live

2. ellie
    - https://ellie-app.com

## Start

### Create new Elm project

    elm init

### Fast forward ...

    git clone git@github.com:jirisliva/elm-workshop.git

    cd 1_hello

    elm make src/Main.elm
